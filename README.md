# Russian-English Translator App

Universal translator from any language using Google Translate API.

![Translator][def]

## Features

- Translation into Russian / English
- Simple and intuitive GUI interface
- Real-time language detection using langdetect

## Requirements

- Internet connection

## Installation

1. [Download latest release](https://github.com/btrf/translator/releases/latest)
2. Run the application by executing: `Translator.exe`

## Usage

1. Type or paste text in any language in the left text area
2. Click "→ To English" to translate the text to English
3. Click "→ To Russian" to translate the text to Russian
4. Use "Swap" to exchange the source and translated text
6. Use "Clear All" to empty both text areas
7. Use "Copy" button to copy translated text to clipboard

## Notes

- The app uses Google Translate API through the googletrans library
- Language detection is automatic using the langdetect library
- Translation quality depends on Google Translate service
